## ✨ About This Project

This project started as a Frontend Mentor challenge, but I turned it into a personalized profile hub that reflects my style and approach to frontend development.

Instead of just replicating the design, I focused on:
- Clean structure
- Thoughtful spacing and typography
- A simple but polished user experience

---

I enjoy building interfaces that are:

Even small projects like this are an opportunity to practice attention to detail and consistency.

## 📚 What I Practiced

- Creating reusable styles instead of one-off CSS
- Handling responsive layouts without overcomplicating things
